## 关键术语速查

| 术语 | 说明 |
|------|------|
| **商业模式** | 产品创造价值、传递价值、获取价值的方式 |
| **订阅制** | 用户按月/年付费订阅使用产品的商业模式 |
| **按量付费** | 用户按使用量（如 Token 数）付费的商业模式 |
| **增值服务** | 基础功能免费，高级功能付费的商业模式 |
| **企业定制** | 为企业客户提供定制化服务的商业模式 |
| **定价策略** | 确定产品价格的方法，包括成本导向、价值导向、竞争导向 |
| **成本导向定价** | 基于成本确定价格的定价策略 |
| **价值导向定价** | 基于用户感知价值确定价格的定价策略 |
| **竞争导向定价** | 基于竞争对手价格确定价格的定价策略 |
| **动态定价** | 根据市场情况动态调整价格的定价策略 |
| **免费+付费** | 基础功能免费，高级功能付费的付费模式 |
| **分层付费** | 不同层级用户付费不同价格的付费模式 |
| **按 Token 计费** | 用户按 Token 使用量付费的付费模式 |
| **企业授权** | 企业购买授权使用产品的付费模式 |
| **Token 成本** | AI 产品的核心成本，每次对话消耗的 Token 成本 |
| **推理成本** | AI 产品的推理计算成本 |
| **存储成本** | AI 产品的数据存储成本 |
| **运维成本** | AI 产品的运维维护成本 |
| **ROI** | Return on Investment，投资回报率 |
| **商业化路径** | 产品从免费到付费的转化路径 |

## 学习建议

1. **先理解商业模式**：商业模式设计是核心，理解订阅制、按量付费、增值服务、企业定制四种模式
2. **掌握定价策略**：定价策略是关键，理解成本导向、价值导向、竞争导向、动态定价四种策略
3. **重点突破成本控制**：AI 产品的成本控制是商业化的核心挑战，Token 成本是关键
4. **动手实践**：设计一个 AI 产品的商业模式方案，计算成本和 ROI
5. **关注 ROI 分析**：ROI 分析是商业化决策的核心方法，需要建立 ROI 模型
6. **参考案例**：商业化案例分析是最佳学习素材，参考 ChatGPT Plus、Claude Pro 等案例
7. **路径思维**：商业化路径设计是长期工作，需要设计免费增长、付费转化、企业销售的完整路径
