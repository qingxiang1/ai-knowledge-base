## 📚 项目简介

这是一个面向AI产品经理的知识体系仓库，包含从基础入门到进阶实战的完整学习路径。通过系统化的知识整理，帮助产品经理快速掌握AI领域的核心能力。

## 🚀 部署指南

### 部署到 Gitee Pages

1. **创建仓库**: 在 Gitee 创建仓库 `ai-project-manager`
2. **推送代码**: 将本仓库内容推送到 Gitee
3. **开启 Pages**: 
   - 进入仓库 → 服务 → Gitee Pages
   - 选择 `master` 分支
   - 选择 `index.html` 为入口文件
   - 点击「启动」

### 本地预览

```bash
# 使用 Python 启动简易服务器
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 访问 http://localhost:8000
```
