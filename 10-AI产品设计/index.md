## 关键术语速查

| 术语 | 说明 |
|------|------|
| **确定性/不确定性** | AI 产品设计的核心原则，AI 输出具有不确定性，需要设计确定性体验 |
| **延迟/响应** | AI 产品设计的核心原则，AI 响应有延迟，需要设计实时反馈机制 |
| **错误/修正** | AI 产品设计的核心原则，AI 可能出错，需要设计错误处理和修正机制 |
| **Copilot** | 副驾驶模式，AI 作为用户的辅助工具，嵌入用户工作流程 |
| **语义搜索** | AI 搜索的核心能力，基于语义理解而非关键词匹配的搜索 |
| **多轮对话** | AI 客服的核心能力，支持多轮交互的对话系统 |
| **意图识别** | AI 客服的核心能力，识别用户意图并匹配对应服务 |
| **人工转接** | AI 客服的核心能力，AI 无法处理时转接人工客服 |
| **知识库问答** | AI 知识库的核心能力，基于企业知识库的问答系统 |
| **语义检索** | AI 知识库的核心能力，基于语义理解的文档检索 |
| **自主决策** | Agent 的核心能力，Agent 自主决策下一步行动 |
| **工具调用** | Agent 的核心能力，Agent 调用外部工具完成任务 |
| **人机协作** | Agent 的核心能力，Agent 与人类协作完成任务 |

## 学习建议

1. **先理解原则**：AI 产品设计原则是核心，理解确定性/不确定性、延迟/响应、错误/修正三大原则
2. **掌握用户体验**：AI 用户体验设计是关键，理解交互模式、反馈机制、信任建立
3. **重点突破 Copilot**：Copilot 是当前主流的 AI 产品形态，是必学内容
4. **动手实践**：分析一个 AI 产品的设计优缺点，理解设计原则的应用
5. **关注 Agent**：Agent 是 AI 产品的进阶形态，需要理解自主决策、工具调用、人机协作
6. **场景思维**：不同场景选择不同产品形态，Copilot vs Agent vs AI客服各有优劣
7. **迭代优化**：AI 产品设计需要持续迭代，建立用户反馈和数据分析机制
