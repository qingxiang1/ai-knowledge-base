## 二、核心模型详解

### 2.1 Qwen2 / Qwen2.5

```
Qwen2.5 概览：

发布：2024年9月
定位：全尺寸开源基座模型

模型矩阵：
┌─────────────────────────────────────────┐
│  轻量版（端侧/边缘）                     │
│  ├── Qwen2.5-0.5B                       │
│  │   └── 手机流畅运行，简单任务          │
│  ├── Qwen2.5-1.5B                       │
│  │   └── 平衡性能，端侧首选              │
│  └── Qwen2.5-3B                         │
│      └── 更强能力，平板/笔记本          │
├─────────────────────────────────────────┤
│  标准版（企业应用）                      │
│  ├── Qwen2.5-7B                         │
│  │   └── 开发者首选，消费级GPU           │
│  ├── Qwen2.5-14B                        │
│  │   └── 性能与效率平衡                  │
│  └── Qwen2.5-32B                        │
│      └── 企业级复杂任务                 │
├─────────────────────────────────────────┤
│  旗舰版（高性能）                        │
│  └── Qwen2.5-72B                        │
│      └── 对标GPT-4，云端部署            │
└─────────────────────────────────────────┘

核心升级（vs Qwen2）：
1. 训练数据：18T tokens（+8T）
2. 指令遵循：提升40%
3. 长文本：128K上下文
4. 多语言：29种语言
5. 代码能力：HumanEval 90%+
6. 数学能力：GSM8K 90%+

性能表现（Qwen2.5-72B）：
| 基准测试 | Qwen2.5-72B | GPT-4 | Llama3.1-70B |
|---------|------------|-------|--------------|
| MMLU | 86.1% | 86.4% | 86.0% |
| GSM8K | 93.2% | 92.0% | 89.5% |
| HumanEval | 86.6% | 90.2% | 82.6% |
| C-Eval | 91.6% | - | 83.0% |
| CMMLU | 91.4% | - | - |
```

### 2.2 Qwen2.5-Coder

```
Qwen2.5-Coder 概览：

发布：2024年11月
定位：代码专用开源模型

模型版本：
- 1.5B / 7B / 14B / 32B
- 基于Qwen2.5架构优化

核心能力：
├── 代码生成：90+编程语言
├── 代码补全：跨文件上下文感知
├── 代码审查：Bug检测、优化建议
├── 代码解释：复杂逻辑分析
├── 测试生成：单元测试、集成测试
└── 代码翻译：跨语言转换

性能表现：
| 基准测试 | Qwen2.5-Coder-32B | GPT-4o | Claude 3.5 Sonnet |
|---------|------------------|--------|-------------------|
| HumanEval | 85.4% | 90.2% | 92.0% |
| MBPP | 81.2% | 85.1% | 87.3% |
| LiveCodeBench | 35.6% | 42.1% | 38.9% |
| MultiPL-E | 78.3% | 82.5% | 80.1% |

特色功能：
- 支持GitHub Copilot兼容格式
- 支持Cursor等IDE集成

- 支持代码仓库级理解
```

- 智能文档处理
