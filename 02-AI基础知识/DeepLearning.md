## 一、深度学习概述

### 1.1 什么是深度学习

**深度学习（Deep Learning）** 是机器学习的一个子领域，它使用多层神经网络（通常包含三层以上的隐藏层）来学习数据的层次化表示。深度学习的"深度"指的是神经网络中隐藏层的数量。

```
机器学习 vs 深度学习：

Machine Learning
│
├── 传统机器学习（SVM、决策树、随机森林）
│   └── 需要人工设计特征
│
└── 深度学习（神经网络）
    └── 自动学习特征表示
```

### 1.2 深度学习的核心优势

| 优势 | 说明 | 对比传统ML |
|------|------|-----------|
| **自动特征学习** | 从原始数据中自动提取层次化特征 | 传统ML需要人工设计特征 |
| **端到端学习** | 直接从输入到输出，减少中间环节 | 传统ML需要多步骤处理 |
| **表达能力** | 深层网络可以逼近任意复杂函数 | 浅层模型表达能力有限 |
| **泛化能力** | 在大规模数据上表现优异 | 小数据上可能过拟合 |

### 1.3 深度学习的三大驱动力

```
深度学习崛起的三大支柱：

    大数据              算力              算法
      │                 │                │
      ▼                 ▼                ▼
  ┌─────────┐      ┌─────────┐      ┌─────────┐
  │ 互联网  │      │  GPU    │      │ 反向传播│
  │ 数据爆炸│  +   │ 并行计算│  +   │ 激活函数│
  │ 标注数据│      │  云算力 │      │ 优化算法│
  └─────────┘      └─────────┘      └─────────┘
```

## 二、神经网络基础

### 2.1 生物神经元 vs 人工神经元

```
生物神经元：

    树突（输入）
       │
       ▼
  ┌─────────┐
  │  细胞体  │ ──→ 轴突（输出）
  │ （处理） │
  └─────────┘

人工神经元（感知机）：

  x₁ ──→
  x₂ ──→   ┌─────────┐
  x₃ ──→   │ 加权求和  │ ──→ 激活函数 ──→ 输出
  ... ──→  │  + 偏置  │
  xₙ ──→   └─────────┘
```

### 2.2 人工神经元的工作原理

```
神经元计算过程：

输入：x₁, x₂, ..., xₙ
权重：w₁, w₂, ..., wₙ
偏置：b

1. 加权求和：z = w₁x₁ + w₂x₂ + ... + wₙxₙ + b

2. 激活函数：a = f(z)

输出：a
```

### 2.3 激活函数

激活函数为神经网络引入非线性，使其能够学习复杂的模式。

| 激活函数 | 公式 | 图像 | 特点 |
|---------|------|------|------|
| **Sigmoid** | f(x) = 1/(1+e⁻ˣ) | S型曲线 | 输出(0,1)，适合二分类，但梯度消失 |
| **Tanh** | f(x) = (eˣ-e⁻ˣ)/(eˣ+e⁻ˣ) | S型曲线 | 输出(-1,1)，零中心化，仍有梯度消失 |
| **ReLU** | f(x) = max(0,x) | 折线 | 计算简单，缓解梯度消失，但可能"死亡" |
| **Leaky ReLU** | f(x) = max(αx, x) | 折线（负区小斜率） | 解决ReLU死亡问题 |
| **Softmax** | f(xᵢ) = eˣⁱ/Σeˣʲ | 概率分布 | 多分类输出层，输出概率和为1 |

```
ReLU函数（最常用）：

    f(x)
    │
    │      ╱
    │    ╱
    │  ╱
    │╱
    └──────────→ x
      0

特点：
- x > 0 时，f(x) = x（线性）
- x ≤ 0 时，f(x) = 0（截断）
- 计算简单，梯度不会消失（正区间）
```

## 三、神经网络架构

### 3.1 前馈神经网络（Feedforward Neural Network）

#### 基本结构

```
多层感知机（MLP）结构：

输入层        隐藏层1       隐藏层2       输出层

x₁ ──→     ┌───┐        ┌───┐        ┌───┐
x₂ ──→ ──→ │ ○ │   ──→  │ ○ │   ──→  │ ○ │ ──→ ŷ
x₃ ──→     │ ○ │        │ ○ │        └───┘
... ──→    │ ○ │        │ ○ │
xₙ ──→     └───┘        └───┘

    ↑           ↑            ↑
  特征输入    特征提取      特征提取    预测输出
```

#### 层的作用

| 层级 | 作用 | 示例 |
|------|------|------|
| **输入层** | 接收原始数据 | 图像像素、文本词向量 |
| **隐藏层** | 提取层次化特征 | 边缘→纹理→形状→物体 |
| **输出层** | 产生最终预测 | 分类概率、回归数值 |

#### 深度 vs 宽度

```
网络设计权衡：

深层网络（Deep）              宽层网络（Wide）

  ○                            ○ ○ ○ ○ ○
  ○                            ○ ○ ○ ○ ○
  ○                            ○ ○ ○ ○ ○
  ○                            ○ ○ ○ ○ ○
  ○                            ○ ○ ○ ○ ○

优势：层次化特征学习          优势：记忆能力强
劣势：训练困难、梯度问题       劣势：容易过拟合
```

### 3.2 卷积神经网络（CNN）

#### 核心思想

**卷积神经网络（Convolutional Neural Network）** 专门用于处理具有网格结构的数据（如图像），通过卷积操作提取局部特征。

```
CNN核心操作：卷积

输入图像：              卷积核（3×3）：
┌───┬───┬───┬───┐      ┌───┬───┬───┐
│ 1 │ 2 │ 3 │ 4 │      │ 1 │ 0 │ 1 │
├───┼───┼───┼───┤      ├───┼───┼───┤
│ 5 │ 6 │ 7 │ 8 │  *   │ 0 │ 1 │ 0 │  =  特征图
├───┼───┼───┼───┤      ├───┼───┼───┤
│ 9 │10 │11 │12 │      │ 1 │ 0 │ 1 │
├───┼───┼───┼───┤      └───┴───┴───┘
│13 │14 │15 │16 │
└───┴───┴───┴───┘

卷积核在图像上滑动，计算局部区域的特征响应
```

#### CNN架构组成

| 组件 | 作用 | 类比 |
|------|------|------|
| **卷积层（Convolution）** | 提取局部特征 | 边缘检测器 |
| **激活层（Activation）** | 引入非线性 | 决策开关 |
| **池化层（Pooling）** | 降维、平移不变性 | 信息压缩 |
| **全连接层（FC）** | 综合特征做决策 | 分类器 |

```
CNN特征学习层次（以图像分类为例）：

输入图像
    │
    ▼
卷积层1 ──→ 边缘特征（横线、竖线、斜线）
    │
    ▼
卷积层2 ──→ 纹理特征（网格、斑点）
    │
    ▼
卷积层3 ──→ 模式特征（眼睛、鼻子）
    │
    ▼
卷积层4 ──→ 物体部件（脸、轮子）
    │
    ▼
全连接层 ──→ 完整物体（人、车、猫）
```

#### 经典CNN架构

| 架构 | 年份 | 特点 | 影响 |
|------|------|------|------|
| LeNet | 1998 | 5层，手写数字识别 | CNN开山之作 |
| AlexNet | 2012 | 8层，ReLU+Dropout | 深度学习复兴标志 |
| VGGNet | 2014 | 16-19层，小卷积核 | 证明深度的重要性 |
| ResNet | 2015 | 152+层，残差连接 | 解决深层网络训练问题 |

### 3.3 循环神经网络（RNN）

#### 核心思想

**循环神经网络（Recurrent Neural Network）** 专门用于处理序列数据（如文本、时间序列），通过循环结构记忆历史信息。

```
RNN结构：

        x₁        x₂        x₃
        │         │         │
        ▼         ▼         ▼
  ┌──→ [A]  ──→ [A]  ──→ [A] ──┐
  │     │         │         │    │
  │     ▼         ▼         ▼    │
  │     h₁        h₂        h₃   │
  │     │         │         │    │
  │     ▼         ▼         ▼    │
  │     ŷ₁        ŷ₂        ŷ₃   │
  │                              │
  └──────── h（隐藏状态）─────────┘

[A]：同一个网络单元，在不同时间步复用
h：隐藏状态，携带历史信息
```

#### RNN的变体

| 变体 | 特点 | 解决的问题 | 应用场景 |
|------|------|-----------|---------|
| **LSTM** | 长短期记忆网络，门控机制 | 长期依赖问题 | 机器翻译、文本生成 |
| **GRU** | 门控循环单元，简化版LSTM | 长期依赖问题 | 语音识别、情感分析 |
| **双向RNN** | 同时考虑过去和未来信息 | 单向信息局限 | 命名实体识别 |

```
LSTM核心：门控机制

        遗忘门 ──→ 决定丢弃哪些历史信息
           │
        输入门 ──→ 决定存储哪些新信息
           │
        输出门 ──→ 决定输出哪些信息
           │
        细胞状态 ──→ 长期记忆的载体
```

### 3.4 神经网络架构对比

| 架构 | 数据类型 | 核心操作 | 典型应用 |
|------|---------|---------|---------|
| **MLP** | 表格数据 | 全连接 | 结构化数据预测 |
| **CNN** | 网格数据（图像） | 卷积 | 图像分类、目标检测 |
| **RNN/LSTM** | 序列数据 | 循环 | 文本、语音、时间序列 |
| **Transformer** | 序列数据 | 注意力 | NLP、多模态（见Transformer章节） |

## 四、神经网络训练

### 4.1 损失函数

损失函数衡量模型预测与真实值之间的差距，训练目标是最小化损失函数。

| 损失函数 | 公式 | 适用场景 |
|---------|------|---------|
| **均方误差（MSE）** | (1/n)Σ(y-ŷ)² | 回归问题 |
| **交叉熵损失（Cross-Entropy）** | -Σy·log(ŷ) | 分类问题 |
| **对比损失（Contrastive）** | 拉近正样本，推远负样本 | 表示学习、Siamese网络 |

### 4.2 反向传播算法

```
反向传播（Backpropagation）流程：

1. 前向传播：输入 → 网络计算 → 得到预测输出
2. 计算损失：预测输出 vs 真实标签
3. 反向传播：从输出层向输入层，逐层计算梯度
4. 参数更新：使用梯度下降更新权重

        前向传播              反向传播

输入 ──→ [Layer1] ──→ [Layer2] ──→ [Layer3] ──→ 输出
                              ↑                    │
                              └────────────────────┘
                                    计算梯度
```

### 4.3 优化器

| 优化器 | 特点 | 适用场景 |
|--------|------|---------|
| **SGD** | 最基础，随机梯度下降 | 简单问题 |
| **Momentum** | 加入动量，加速收敛 | 存在局部最优 |
| **Adam** | 自适应学习率，最常用 | 大多数场景 |
| **AdamW** | Adam + 权重衰减 | 大模型训练 |

### 4.4 正则化技术

正则化用于防止过拟合，提高模型泛化能力。

| 技术 | 原理 | 实现方式 |
|------|------|---------|
| **Dropout** | 随机丢弃神经元 | 训练时以概率p关闭神经元 |
| **L2正则化** | 限制权重大小 | 损失函数中加入权重平方和 |
| **Batch Normalization** | 标准化层输入 | 每层输入归一化 |
| **Early Stopping** | 提前停止训练 | 验证集性能不再提升时停止 |

```
Dropout示意：

训练时（Dropout=0.5）：              推理时：

  ○   ○   ○   ○                      ○   ○   ○   ○
  │   │   ✕   │                      │   │   │   │
  ○   ✕   ○   ○                      ○   ○   ○   ○
  │   │   │   ✕                      │   │   │   │
  ○   ○   ○   ○                      ○   ○   ○   ○

（✕表示被丢弃的神经元，输出乘以0.5）
```

## 七、本章小结

```
深度学习核心知识框架：

Deep Learning
│
├── 基础
│   ├── 人工神经元（加权求和 + 激活函数）
│   ├── 激活函数（ReLU/Sigmoid/Softmax）
│   └── 网络结构（输入层-隐藏层-输出层）
│
├── 架构
│   ├── CNN：局部连接，权值共享，适合图像
│   ├── RNN/LSTM：循环结构，适合序列
│   └── Transformer：注意力机制（详见Transformer章节）
│
├── 训练
│   ├── 损失函数（MSE/交叉熵）
│   ├── 反向传播（梯度计算与传播）
│   ├── 优化器（Adam/SGD）
│   └── 正则化（Dropout/BatchNorm）
│
└── 应用
    ├── 计算机视觉（分类/检测/生成）
    ├── 自然语言处理（翻译/生成/理解）
    └── 推荐系统/广告/游戏AI
```

**关键认知**：
1. 深度学习的核心是深层神经网络，自动学习层次化特征
2. CNN适合图像，RNN适合序列，Transformer适合长序列和并行计算
3. 训练神经网络需要大量数据、算力和调参经验
4. 过拟合是深度学习的主要挑战，需要正则化技术
5. AI产品经理需要理解原理和适用场景，而非实现细节
