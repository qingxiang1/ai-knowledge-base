## 关键术语速查

| 术语 | 说明 |
|------|------|
| **竞品分析** | 分析竞争对手产品的定位、功能、用户体验、商业模式、优劣势 |
| **竞品选择** | 确定需要分析的竞品，包括直接竞品、间接竞品、潜在竞品 |
| **分析维度** | 竞品分析的维度，包括产品定位、核心功能、用户体验、商业模式、技术架构 |
| **数据采集** | 竞品数据采集方法，包括公开数据、用户反馈、行业报告、第三方数据 |
| **分析框架** | 竞品分析的结构化框架，如 SWOT 分析、波特五力分析、价值链分析 |
| **竞品监控** | 持续监控竞品的动态，包括产品更新、市场动态、用户反馈 |
| **差异化定位** | 与竞品形成差异化的产品定位，避免同质化竞争 |
| **竞争策略** | 与竞品竞争的策略，包括进攻策略、防御策略、合作策略 |
| **防御策略** | 防止竞品侵蚀市场份额的策略，如建立壁垒、提升用户体验 |
| **进攻策略** | 主动进攻竞品市场份额的策略，如差异化功能、价格战 |
| **SWOT 分析** | 分析竞品的优势、劣势、机会、威胁 |
| **波特五力分析** | 分析行业竞争的五种力量：供应商、买家、竞品、替代品、潜在进入者 |
| **价值链分析** | 分析竞品的价值创造过程，找到差异化机会 |

## 学习建议

1. **先理解方法**：竞品分析方法论是核心，理解竞品选择、分析维度、数据采集、分析框架
2. **重点突破主流竞品**：ChatGPT/Claude/Copilot 是当前主流 AI 产品，是必学内容
3. **动手实践**：分析一个 AI 产品的竞品，建立竞品分析框架
4. **关注数据监控**：竞品数据监控是长期工作，需要建立监控体系
5. **制定策略**：竞品分析的最终目的是制定竞争策略，差异化定位是关键
6. **参考案例**：主流 AI 产品的竞品分析是最佳学习素材
7. **持续更新**：AI 产品迭代快，竞品分析需要持续更新
