## 关键术语速查

| 术语 | 说明 |
|------|------|
| **项目实战** | 完成一个 AI 产品从需求分析到上线运营的全流程实践 |
| **项目选择** | 选择适合实战的项目，考虑技术难度、业务价值、学习价值 |
| **需求分析** | 分析项目的用户需求、业务需求、技术需求 |
| **技术选型** | 选择项目的技术栈，包括模型、框架、平台、工具 |
| **开发流程** | 项目的开发流程，包括设计、开发、测试、上线 |
| **上线运营** | 项目上线后的运营工作，包括监控、优化、迭代 |
| **项目复盘** | 复盘项目的经验教训，总结成功经验和失败教训 |
| **复盘流程** | 项目复盘的流程，包括回顾、分析、总结、应用 |
| **复盘维度** | 项目复盘的维度，包括需求、设计、开发、运营 |
| **复盘输出** | 项目复盘的输出，包括经验文档、教训文档、改进方案 |
| **项目案例库** | 收集整理项目案例，包括行业案例、企业案例、开源案例 |

## 学习建议

1. **先理解方法**：项目实战方法论是核心，理解项目选择、需求分析、技术选型、开发流程、上线运营
2. **重点突破典型产品**：AI客服/AI知识库/Copilot 是当前主流 AI 产品，是必学内容
3. **动手实践**：完成一个 AI 产品项目实战，理解全流程
4. **关注项目复盘**：项目复盘是提升的关键，需要总结经验教训
5. **参考案例**：项目案例库是最佳学习素材，参考行业案例、企业案例
6. **持续迭代**：项目实战需要持续迭代，建立项目迭代机制
7. **作品集思维**：项目实战是作品集的核心内容，需要建立项目作品集
